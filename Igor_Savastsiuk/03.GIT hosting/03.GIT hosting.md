##### 2. Script for pushing

###### Script:

```bash
#!/bin/bash
repos=(github gitlab bitbucket)


for i in ${repos[@]}; do
	git push -u $i --all
	echo -e "\033[36;1m**********************$i pushed**********************\033[0m"
	sleep 1
done
```
